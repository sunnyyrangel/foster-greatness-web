# Foster Greatness Website - Project Overview

## Purpose
Unified Next.js website for Foster Greatness, a lived experience-led nonprofit platform creating lifelong community and belonging for current and former foster youth nationwide.

The site consolidates 7 separate initiatives into one cohesive platform:
- Community Updates (homepage)
- Holiday Gift Drive 2025
- Gingerbread Contest
- Storytellers Collective
- Meal Kit Sponsors
- Newsletter Widget
- Circle Events

## Key Features
- **Configuration-driven architecture** - Content and campaigns managed through data files (not scattered code)
- **Campaign management system** - Centralized campaign definitions with visibility flags
- **Multiple page types** - Full site pages with navigation + embeddable widgets
- **API integrations** - Beehiiv (newsletter), Circle.so (events), Supabase (gift drive), Stripe (donations)
- **Analytics** - Vercel Web Analytics for tracking

## Project Type
Production nonprofit website with active campaigns and real user interactions.

## Development Platform
- Deployed on Vercel
- Repository: GitHub
- System: macOS (Darwin)
