## GEO (AI Engine) Optimization Checklist

For every new page, ensure:

- [ ] **Statistics with sources**: Include 3-5 specific stats with citations
- [ ] **Expert quotes**: Include quotes from team members with credentials
- [ ] **Clear definitions**: Define key terms in first paragraph
- [ ] **FAQ section**: Add 4-6 FAQs with schema markup
- [ ] **Last updated date**: Show content freshness
- [ ] **Structured data**: Implement relevant JSON-LD schema
- [ ] **Authoritative sources**: Link to .gov and .edu sources
